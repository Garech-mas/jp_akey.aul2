# AviUtl2 アクセスキー付き日本語リソース
AviUtl2の各メニュー・一部ボタン項目にアクセスキーを追加する言語ファイル(`.aul2`)です。

AviUtl ExEdit2 beta35 までの項目に対応しています。

## ダウンロード・インストール
https://raw.githubusercontent.com/Garech-mas/jp_akey.aul2/refs/heads/main/日本語(アクセスキー付き).org.aul2

上記のリンクを右クリックし、**名前を付けてリンク先を保存**から`日本語(アクセスキー付き).org.aul2` をダウンロードできます。

AviUtl2を起動し、ダウンロードしたファイルをプレビュー画面にドラッグ＆ドロップすることで導入できます。

自動で適用されない場合、メニューバー 設定 > 言語の設定 から、`日本語(アクセスキー付き)`を選択して適用してください。

## 導入・更新時の注意点
リリース時最新版（beta35）では、言語ファイルを変更すると**ショートカットキーの設定が失われます。**

ショートカットキーの設定を変更している方は、導入・更新後に再設定してください。
> [!NOTE]
> 混乱を避けるため、上記問題が修正されるまでは、AviUtl2 カタログへの更新登録を行わないようお願いいたします。

## カスタマイズ
オブジェクト追加メニューのラベル名の末尾に、`(&X)`のように記載を追加することで、手動でアクセスキーを追加することができます。

手動でアクセスキーを変更・追加したファイルを、`日本語(アクセスキー付き).aul2` のようにリネームして配置し直してください。<br>
(`.org`部分を削除すればOKです。)
